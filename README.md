# Painel de Vendas — Depósito J Martins

Uma aplicação interativa desenvolvida em **Python** e **Streamlit** para análise de dados e visualização de vendas, criada especificamente para otimizar a gestão e a tomada de decisões do **Depósito J Martins**.

Este projeto foi desenvolvido como parte de um **Trabalho de Extensão Universitária**, unindo o conhecimento acadêmico às necessidades reais do mercado.

## 🔗 Links do Projeto

*  **Aplicação em produção:** https://painelvendasjmartins.streamlit.app/
*  **Repositório GitHub:** https://github.com/TLP7563/Analise-Dados

---

## Objetivo do Projeto

O objetivo principal desta ferramenta é transformar dados brutos de vendas do Depósito J Martins em insights visuais e acionáveis.

Por meio de dashboards intuitivos, a gerência pode:

* Acompanhar indicadores de desempenho (KPIs);
* Identificar gargalos operacionais;
* Entender o comportamento dos clientes;
* Planejar compras e reposição de estoque com maior eficiência;
* Apoiar a tomada de decisões baseada em dados.

---

## Funcionalidades Principais

### Dashboards Interativos

Visualização dinâmica de métricas como:

* Faturamento;
* Volume de vendas;
* Ticket médio;
* Evolução de desempenho.

### Filtros Personalizados

Permite análises por:

* Período;
* Categoria de produto;
* Vendedor;
* Região.

### Análise de Tendências

Gráficos que identificam:

* Crescimento das vendas;
* Sazonalidade;
* Padrões de consumo.

### Gestão de Estoque

Insights sobre:

* Produtos mais vendidos;
* Curva ABC;
* Apoio à reposição de mercadorias.

### Interface Intuitiva

Desenvolvida para facilitar o uso por gestores e colaboradores, sem necessidade de conhecimentos técnicos avançados.

## Como Executar o Projeto Localmente

### Pré-requisitos

* Python 3.8 ou superior
* Git instalado na máquina

### 1. Clonar o Repositório

```bash
git clone https://github.com/TLP7563/Analise-Dados.git
cd Analise-Dados
```

### 2. Criar Ambiente Virtual (Opcional)

```bash
python -m venv venv
```

### 3. Ativar o Ambiente Virtual

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 4. Instalar Dependências

```bash
pip install -r requirements.txt
```

### 5. Executar a Aplicação

```bash
streamlit run app.py
```

A aplicação será aberta automaticamente no navegador.
